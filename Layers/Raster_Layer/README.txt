This folder has the raster implemented in the article to classify all Navarra's plots

CCDC_X_Ty_name

X means: C: control, A: Forest Harvesting practices

Ty: indicate the level of tBreak

name: show the bands involve. B2aB7 means that all bands were tested in CCDC.

*CCDC_raster are the results of CCDC algorithm after process the different levels of changes.
*The raster with the name _ext are those used to mask plots in different code (There are into Extension.zip)
